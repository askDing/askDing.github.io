    多媒体

    动态文字

    <marquee direction="left/right/up/down" behavior="scroll/slide/alternate" loop=1 scrollamount=5 scrolldelay=1 bgcolor=“#FFFF55” width=300 height=500> 动态文字. </marquee>

direction: 滚动方向
behavior: 滚动方式
loop: 循环次数
scrollamount: 滚动速度
scrolldelay: 滚动延迟
bgcolor: 背景颜色
width、height:滚动面积
